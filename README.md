## 📖 Overview

HostelMate is a web-based Hostel Management System developed using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

The application provides a centralized platform for both **Students** and **Administrators** to efficiently manage hostel-related activities.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/BHARGAV3009/HostelMate.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the **backend** folder.

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

> **Do not commit `.env` files or any secrets to GitHub.**
